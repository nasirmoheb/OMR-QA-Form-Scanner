QUICK START
───────────────────────────────────────────────────────────────────

1. Extract this ZIP file to any folder (e.g., C:\OMR_Scanner)
2. Open the extracted folder
3. Double-click "OMR_Scanner.exe" to run
4. No installation required!


FIRST RUN
───────────────────────────────────────────────────────────────────

Windows may show a security warning:
  "Windows protected your PC"

This is normal for unsigned applications.

To run:
  1. Click "More info"
  2. Click "Run anyway"

This warning only appears once.


SYSTEM REQUIREMENTS
───────────────────────────────────────────────────────────────────

✓ Windows 10 (64-bit) or Windows 11
✓ 4 GB RAM (8 GB recommended)
✓ 250 MB free disk space
✓ 1280x720 display or higher


FEATURES
───────────────────────────────────────────────────────────────────

✓ Create and manage QA survey forms

USAGE
───────────────────────────────────────────────────────────────────

1. CREATE SURVEY
   - Click "New Survey" button
   - Fill in survey details (faculty, professor, subject, etc.)
   - Save the survey

2. PRINT FORM
   - Select survey from dashboard
   - Click "Print" button
   - PDF form opens in browser
   - Print and distribute to students

3. SCAN FORMS
   - Collect filled forms from students
   - Scan forms (300 DPI recommended, color mode)
   - Save scanned images to a folder

4. PROCESS FORMS
   - Select survey from dashboard
   - Click "Process" button
   - Choose folder with scanned images
   - Wait for processing to complete

5. VIEW RESULTS
   - Click "Results" button on processed survey
   - View interactive Dari report
   - Analyze statistics and charts
   - Export or print report


TIPS FOR BEST RESULTS
───────────────────────────────────────────────────────────────────

✓ Scan forms in COLOR mode (not grayscale)
✓ Use 300 DPI resolution
✓ Ensure forms are flat and well-lit
✓ Use blue or black pen for filling
✓ Mark checkboxes clearly (fill completely)
✓ Avoid stray marks or smudges


CREATING DESKTOP SHORTCUT
───────────────────────────────────────────────────────────────────

1. Right-click "OMR_Scanner.exe"
2. Select "Send to" → "Desktop (create shortcut)"
3. Shortcut appears on desktop


TROUBLESHOOTING
───────────────────────────────────────────────────────────────────

Problem: Application won't start
Solution: Install Visual C++ Redistributables
  Download: https://aka.ms/vs/17/release/vc_redist.x64.exe

Problem: Slow first startup
Solution: Wait 30-60 seconds (Windows Defender scanning)

Problem: Database errors
Solution: Run as administrator (right-click → Run as administrator)

Problem: Forms not detected
Solution: 
  - Ensure scanner is in COLOR mode
  - Check scan quality (300 DPI)
  - Verify forms are properly aligned


DATA LOCATION
───────────────────────────────────────────────────────────────────

All data is stored in the "data" folder:
  - data/omr.db (SQLite database)

To backup your data:
  - Copy the entire "data" folder


UPDATING
───────────────────────────────────────────────────────────────────

To update to a new version:
  1. Download new version ZIP
  2. Extract to a NEW folder
  3. Copy "data" folder from old version to new version
  4. Run new version


UNINSTALLING

No registry entries or system files are created.
