# سیستم تحلیل تضمین کیفیت تدریس (Tadris QA)

<div dir="rtl">

## 📋 درباره سیستم

سیستم تحلیل تضمین کیفیت تدریس یک نرم‌افزار جامع برای ارزیابی کیفیت تدریس اساتید توسط محصلان در پوهنتون‌ها می‌باشد. این سیستم با استفاده از فناوری OMR (تشخیص نشانه‌های نوری) فورم‌های کاغذی را اسکن و تحلیل می‌کند.

## ✨ ویژگی‌های اصلی

### 📊 تحلیل جامع
- تحلیل خودکار فورم‌های ارزیابی
- تولید گزارش‌های تفصیلی به زبان دری
- نمودارهای بصری (میله‌ای، راداری، دونات)
- تحلیل 14 سوال در 4 بُعد آکادمیک

### 🎨 گزارش‌های حرفه‌ای
- گزارش 2 صفحه‌ای با چیدمان حرفه‌ای
- لوگوهای جاسازی شده (مستقل از فایل‌های خارجی)
- آماده چاپ (دیالوگ چاپ خودکار)
- نام پوهنتون و پوهنحی به صورت پویا

### 🔧 قابلیت‌های فنی
- رابط کاربری گرافیکی (GUI) با CustomTkinter
- پشتیبانی از زبان‌های دری و پشتو
- پایگاه داده SQLite
- صادرات به PDF و CSV
- سازگار با Windows 10/11

## 📦 نصب

### دانلود
[دانلود آخرین نسخه](installer_output/Tadris_QA_Setup_v1.0.0.exe)

### نصب از فایل
1. فایل `Tadris_QA_Setup_v1.0.0.exe` را دانلود کنید
2. روی فایل دوبار کلیک کنید
3. مراحل نصب را دنبال کنید
4. برنامه از منوی Start اجرا کنید

### پیش‌نیازها
- Windows 10 یا 11 (64-bit)
- حداقل 4GB RAM
- 200MB فضای خالی دیسک
- هیچ نصب Python مورد نیاز نیست

## 🎯 نحوه استفاده

### 1. ایجاد سروی جدید
```
1. روی دکمه "سروی جدید" کلیک کنید
2. اطلاعات را وارد کنید:
   - نام پوهنتون
   - نام پوهنحی
   - نام دیپارتمنت
   - نام استاد
   - نام مضمون
   - سمستر و سال تحصیلی
3. روی "ذخیره" کلیک کنید
```

### 2. تولید فورم PDF
```
1. سروی را انتخاب کنید
2. روی "تولید فورم PDF" کلیک کنید
3. فایل PDF را ذخیره کنید
4. فورم را چاپ و به محصلان توزیع کنید
```

### 3. اسکن و تحلیل
```
1. فورم‌های تکمیل شده را اسکن کنید
2. تصاویر را به برنامه اضافه کنید
3. روی "تحلیل" کلیک کنید
4. نتایج را مشاهده کنید
```

### 4. تولید گزارش
```
1. روی "گزارش Dari QA" کلیک کنید
2. گزارش در مرورگر باز می‌شود
3. دیالوگ چاپ خودکار ظاهر می‌شود
4. گزارش را چاپ یا ذخیره کنید
```

## 📊 انواع گزارش‌ها

### گزارش Dari QA (پیشنهادی)
- گزارش کامل 2 صفحه‌ای به زبان دری
- شامل تمام تحلیل‌ها و نمودارها
- آماده چاپ و ارائه

### گزارش تحلیلی HTML
- گزارش تعاملی با نمودارهای Plotly
- قابل مشاهده در مرورگر
- شامل تحلیل‌های آماری

### صادرات CSV
- داده‌های خام برای تحلیل بیشتر
- قابل باز شدن در Excel
- مناسب برای پردازش داده

## 🔧 ساخت از کد منبع

### پیش‌نیازها
```bash
Python 3.10+
pip
virtualenv
```

### مراحل
```powershell
# 1. کلون کردن مخزن
git clone https://github.com/[username]/tadris-qa.git
cd tadris-qa

# 2. ایجاد محیط مجازی
python -m venv venv
.\venv\Scripts\activate

# 3. نصب وابستگی‌ها

# 4. اجرای برنامه
python main.py

# 5. اجرای تست‌ها

```
