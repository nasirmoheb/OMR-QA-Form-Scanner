## ✨ ویژگی‌های جدید

### 1. نام پوهنتون و پوهنحی به صورت پویا
- **قبل**: نام پوهنتون و پوهنحی به صورت ثابت در گزارش نوشته می‌شد
- **اکنون**: نام‌ها از معلومات سروی خوانده می‌شود
- **مزیت**: سیستم برای پوهنتون‌های مختلف قابل استفاده است

```
سرلوحه گزارش:
وزارت تحصیلات عالی - [نام پوهنتون شما]
[نام پوهنحی شما] | کمیته ارتقای کیفیت
```

### 2. گزارش‌ها به صورت خودکار برای چاپ آماده می‌شوند
- پنجره چاپ به صورت خودکار باز می‌شود
- گزارش بلافاصله قابل چاپ است
- زمان کاربر صرفه‌جویی می‌شود

### 3. بهبود چیدمان گزارش
- **صفحه دوم**: حاشیه بالایی 20 میلی‌متر اضافه شد
- **تعداد صفحات**: محتوا در 2 صفحه جای می‌گیرد (به جای 3 صفحه)
- **صرفه‌جویی کاغذ**: 33٪ کاهش مصرف کاغذ

### 4. اعتبار سیستم
اطلاعات توسعه‌دهنده در پایین گزارش اضافه شد:
```
سیستم تحلیل تضمین کیفیت • طراحی و توسعه: نصیر احمد محب • پوهنتون بدخشان
```

**راه‌حل**:

### 3. لوگوها در فورم‌های PDF نمایش داده نمی‌شدند

**راه‌حل**:
- مسیرهای صحیح برای لوگوها در محیط تولیدی
- استفاده از تنظیمات Config
- ✅ برطرف شد

### 4. خطای "اجازه رد شد" هنگام ذخیره گزارش‌ها

## 📦 نصب و راه‌اندازی

### پیش‌نیازها
- Windows 10/11 (64-bit)
- هیچ نصب Python مورد نیاز نیست

### مراحل نصب
1. فایل نصب را دانلود کنید: `Tadris_QA_Setup_v1.0.0.exe`
2. روی فایل دوبار کلیک کنید
3. مراحل نصب را دنبال کنید
4. برنامه از منوی Start اجرا کنید

### ساخت از کد منبع
```powershell
# 1. فعال‌سازی محیط مجازی
.\venv\Scripts\activate

# 2. نصب وابستگی‌ها
pip install -r requirements.txt

# 3. اجرای تست‌ها
.\venv\Scripts\python.exe test_all_fixes.py

# 4. ساخت نصب‌کننده
.\build_installer.ps1
```

```python
# نام پوهنتون

DEFAULT_FACULTY_NAME = "پوهنحی کمپیوتر ساینس"
```

### سفارشی‌سازی
برای تغییر نام پوهنتون یا پوهنحی:
1. هنگام ایجاد سروی، نام صحیح را وارد کنید
2. یا مقادیر پیش‌فرض را در `src/config.py` تغییر دهید
