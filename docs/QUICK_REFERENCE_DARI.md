## 📈 تفسیر نتایج

### نمره کلی
```
90-100٪ = عالی ⭐⭐⭐⭐⭐
80-89٪  = خوب ⭐⭐⭐⭐
70-79٪  = متوسط ⭐⭐⭐
60-69٪  = نیاز به بهبود ⭐⭐
<60٪    = ضعیف ⭐
```

### رنگ‌ها
```
🟢 سبز = عملکرد مطلوب (بلی)
🟡 زرد = متوسط (نسبتاً)
🔴 قرمز = نیاز به بهبود (نخیر)
```

### نمودار راداری
```
هر ضلع = یک بُعد آکادمیک
مساحت بزرگتر = عملکرد بهتر
نقاط ضعف = گوشه‌های کوچک
```
